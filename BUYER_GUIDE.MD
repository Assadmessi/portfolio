# Portfolio Admin – How to Update Projects

## Login
Go to:
https://your-site.com/admin

Login with the email/password provided.

---

## Adding / Editing Projects
1. Open **Projects**
2. Click **Add** or select an existing project
3. Fill in:
   - Title
   - Description
   - Tags
   - Links (optional)

---

## Changing Project Images (Recommended)
We recommend **Cloudinary (FREE)**.

### Steps:
1. Go to https://cloudinary.com
2. Create a free account
3. Upload your image
4. Copy the image URL
5. Paste it into **Thumbnail URL**
6. Click **Save**

The system automatically optimizes the image for desktop and mobile.

⚠️ Do NOT use:
- Google Drive links
- Facebook / Instagram links

---

## Publish Changes
Click **Save** → the public site updates instantly.
